# node-graphql-starter
A simple node.js GraphQL API quickstart

## How to use
1. Open the CLI
2. Enter `git clone https://github.com/matt-sungwook/node-graphql-starter.git myapp`
3. Enter `cd myapp`
4. Start building your GraphQL API!
